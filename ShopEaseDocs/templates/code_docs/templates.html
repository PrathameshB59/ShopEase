ShopEase uses Django's template engine with template inheritance.

Template Inheritance

  • base.html - Main layout with navbar, footer, CSS/JS
  • includes/ - Reusable components (navbar, footer, forms)
  • App templates - Extend base.html with specific content

Template Tags

  • {% verbatim %}{% load static %}{% endverbatim %} - Load static files
  • {% verbatim %}{% url 'name' %}{% endverbatim %} - URL reverse lookup
  • {% verbatim %}{% csrf_token %}{% endverbatim %} - CSRF protection
