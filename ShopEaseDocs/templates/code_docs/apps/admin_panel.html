Admin Panel App

The Admin Panel provides staff management functionality.

Features

  • Dashboard with analytics
  • Product management (CRUD)
  • Order management and status updates
  • User management
  • Reports and exports

Access Control

Uses middleware and decorators:

  • AdminPortAccessMiddleware - Restricts access to staff only
  • @admin_required - View-level protection
  • @permission_required - Granular permissions
