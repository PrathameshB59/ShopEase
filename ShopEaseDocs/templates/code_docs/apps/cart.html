Cart App

The Cart app manages shopping cart functionality.

Features

  • Session-based cart for guests
  • Database-backed cart for logged-in users
  • Cart merging on login
  • Wishlist functionality

CartService

The CartService class handles all cart operations:

  • add_item() - Add product to cart
  • remove_item() - Remove item from cart
  • update_quantity() - Update item quantity
  • get_total() - Calculate cart total
  • merge_with_user_cart() - Merge guest cart with user
