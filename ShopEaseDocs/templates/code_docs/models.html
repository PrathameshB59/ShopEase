ShopEase uses Django's ORM with the following key models:

Product Models

  • Category - Product categories with hierarchy support
  • Product - Main product model with pricing, inventory
  • ProductImage - Multiple images per product
  • Review - Customer reviews and ratings

User Models

  • User - Django's built-in User model
  • Profile - Extended user profile (addresses, phone)

Cart Models

  • Cart - Shopping cart (session or user-based)
  • CartItem - Items in the cart
  • Wishlist - User's saved items

Order Models

  • Order - Customer orders with status tracking
  • OrderItem - Individual items in an order
  • Payment - Payment records (Razorpay integration)
