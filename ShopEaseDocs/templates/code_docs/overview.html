About ShopEase

ShopEase is a full-featured e-commerce platform built with Django. It provides a complete online shopping experience with features like product browsing, cart management, secure checkout, and order tracking.

Architecture

The project follows Django's MVT (Model-View-Template) architecture pattern with a clean separation of concerns:

  • Models - Define the data structure (products, users, orders)
  • Views - Handle business logic and request processing
  • Templates - Render the HTML responses

Multi-Server Setup

ShopEase runs on two separate servers:

  • Customer Server (Port 8000) - Public-facing e-commerce site
  • Admin Server (Port 8080) - Staff-only admin dashboard
