Admin Panel Documentation

Complete guide to the ShopEase custom admin panel

Overview

The ShopEase Admin Panel is a custom-built administrative interface that provides comprehensive management capabilities for your e-commerce platform. Built with Django 5.0 and Bootstrap 5, it offers role-based access control, detailed analytics, and powerful management tools.

Access URL: http://localhost:8000/dashboard/

Requirements

  • Django 5.0+
  • MySQL Database
  • WeasyPrint (for PDF invoices)
  • Bootstrap 5 (included via CDN)
  • Chart.js (included via CDN)

Features

Role-Based Access

5 admin roles with 12 granular permissions

Order Management

Full order lifecycle, refunds, PDF invoices

Product Analytics

View tracking, engagement metrics, scoring

Review Moderation

Approve/reject reviews, bulk actions

User Management

Staff creation, role assignment, activity logs

Reports & Export

Sales, revenue reports with CSV export

Installation & Setup

Step 1: Add to Installed Apps

# config/settings.py
INSTALLED_APPS = [
    ...
    'apps.admin_panel',
]

Step 2: Add Middleware

# config/settings.py
MIDDLEWARE = [
    ...
    'apps.admin_panel.middleware.ProductViewTrackingMiddleware',
]

Step 3: Add Context Processor

# config/settings.py - Inside TEMPLATES OPTIONS
'context_processors': [
    ...
    'apps.admin_panel.context_processors.admin_permissions',
]

Step 4: Add URL Pattern

# config/urls.py
urlpatterns = [
    ...
    path('dashboard/', include('apps.admin_panel.urls', namespace='admin_panel')),
]

Step 5: Run Migrations

python manage.py makemigrations admin_panel
python manage.py migrate

Step 6: Create Superuser

python manage.py createsuperuser

Dashboard

URL: /dashboard/

The main dashboard provides an overview of key business metrics:

  • Today's orders and revenue
  • Pending orders count
  • Low stock alerts
  • Recent orders list
  • Quick navigation to all modules

Order Management

URL: /dashboard/orders/

Features

  • View all orders with filtering (status, date, search)
  • Order detail view with full information
  • Update order status
  • Generate PDF invoices
  • Process refunds with stock restoration
  • Bulk status updates
  • Automatic email notifications

Order Status Flow

PENDING → PROCESSING → SHIPPED → DELIVERED → COMPLETED
Orders can be cancelled at any stage before completion

Product Analytics

URL: /dashboard/products/

Metrics Tracked

  • Page views (total & unique)
  • Cart additions
  • Wishlist additions
  • Purchases
  • Revenue
  • Average rating

Featured Score Formula

Score = Revenue (40%) + Purchases (20%) + Engagement (15%) + Views (10%) + Rating (10%) + Reviews (5%)

Featured Products

URL: /dashboard/products/featured/dashboard/

Features

  • View top products by featured score
  • Auto-suggested products based on analytics
  • Manual product featuring
  • Set priority and expiry dates
  • Track featured product performance

Review Management

URL: /dashboard/reviews/

Features

  • View all reviews with filtering
  • Pending reviews queue
  • Approve/reject individual reviews
  • Bulk approve/reject/delete
  • Review analytics dashboard

User Management

URL: /dashboard/users/

Features

  • View all users (staff and customers)
  • User detail with order/review history
  • Assign/remove admin roles
  • Activate/deactivate accounts
  • Create new staff users
  • Activity log viewer

Reports & Analytics

URL: /dashboard/reports/

Available Reports

Report URL Description
Sales Report /reports/sales/ Orders, revenue, top products
Revenue Analytics /reports/revenue/ Monthly trends, category breakdown
Product Performance /reports/products/ Best sellers, stock alerts
All reports support CSV export for further analysis

API Endpoints

Featured Products Auto-Suggest

GET /dashboard/products/featured/auto-suggest/

Parameters:
- days (int): Number of days to analyze (default: 7)
- limit (int): Number of suggestions (default: 10)

Response:
{
  "success": true,
  "suggestions": [
    {
      "id": 1,
      "name": "Product Name",
      "featured_score": 85.5,
      "revenue": 15000.00,
      ...
    }
  ]
}

Management Commands

Aggregate Analytics

Run daily to calculate featured product scores:

# Run for yesterday (default)
python manage.py aggregate_analytics

# Run for specific date
python manage.py aggregate_analytics --date 2024-01-15

# Schedule with cron (daily at 2 AM)
0 2 * * * cd /path/to/shopease && python manage.py aggregate_analytics

Customization

Adding New Permissions

# 1. Add field to AdminRole model
can_new_permission = models.BooleanField(default=False)

# 2. Update context processor
'can_new_permission': admin_role.can_new_permission,

# 3. Use in templates
{% if user_permissions.can_new_permission %}
    
{% endif %}

# 4. Use in views
@permission_required('can_new_permission')
def my_view(request):
    ...
